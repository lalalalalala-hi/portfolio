Shopping Mall
Map App

An AR-based indoor navigation app designed to enhance the shopping experience by providing real-time directions, store information, and promotional updates within a shopping mall.

Key Feature

  • AR Indoor Navigation
    Users can navigate the mall using AR overlays, guiding them from one location to another with ease.
  • Real-Time Store and Promotion Information
    The app delivers up-to-date information on store locations, ongoing promotions, and events, allowing users to make informed decisions on the go.
  • Admin Dashboard
    A comprehensive web-based interface for mall administrators to monitor user activity, manage inventory, and analyze popular routes.
  • Dynamic Routing
    Personalized routes are generated based on the userâ€™s current location and destination, optimizing their shopping experience.
